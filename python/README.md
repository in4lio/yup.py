### How to use `yupp` with Python 2

* Install Python package:
```
pip install yupp
```

* [Configure Sublime Text](../sublime_text/) (optional)

* Start your code on Python with:
```py
# coding: yupp
```
for example:
```
# coding: yupp
# hello.py

($set msg "Hello world!")
print ($msg)
```

* Run the script in the usual way:
```
>python hello.py
Hello world!
```
