### How to start using `yupp` with Python 2

`1.` Install Python package:
```
pip install yupp
```

`2.` Start your code on Python with:
```
# coding: yupp
```
for example:
```
# coding: yupp
# test.py

($set hola 'Hello world!')
print ($hola)
```

`3.` Run the script in the usual way:
```
python ./test.py

Hello world!
```
